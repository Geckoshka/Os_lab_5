# Лабораторна робота №5: Робота з Git
## Опис: Освоєння основ роботи з системою контролю версій Git та сервісом GitHub.
## Мета: Навчитися створювати репозиторії, працювати з гілками, створювати коміти та взаємодіяти з віддаленим репозиторієм.
## Контрольні питання
1. **Що таке Git?** - Це розподілена система контролю версій.
2. **Що таке коміт?** - Це знімок стану вашого проєкту в певний момент часу.
